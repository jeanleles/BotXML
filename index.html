Bot-XML

Bot-XML é um projeto para ler as informações contidas em um arquivo XML e carregrar essas informações para o front-end do projeto, usando para isso o jQuery. Projeto baseado no demonstrado no vídeo do canal SOS Neurônios [https://youtu.be/4Ud68eOm9Vg].

0 objetivo agora é substituir o jQuery pelas funções nativas do JavaScritp ES6.
